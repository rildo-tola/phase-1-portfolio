# phase-1-portfolio
Task Tracker App

A simple and user-friendly task management application built with vanilla JavaScript.

Features:
• Add and delete tasks
• Mark tasks as completed
• Persistent storage using localStorage
• Dynamic task counter
• Keyboard support (Enter key)
• Accessible and responsive UI with subtle animations

Built with:
HTML, CSS, JavaScript (ES6), Git & GitHub

What I learned:
This project helped me strengthen my understanding of DOM manipulation, event handling, state management, and user experience design. I also gained experience improving accessibility and building interactive interfaces without frameworks.
